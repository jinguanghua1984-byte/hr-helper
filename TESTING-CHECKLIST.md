## 🐛 常见问题排查

### 问题 1：扩展无法加载

**可能原因**：
- manifest.json 路径错误
- popup.html 语法错误

**解决方案**：
```bash
# 检查 manifest.json
cat d:\workspace\hr-helper\manifest.json

# 重新加载扩展
chrome://extensions/ → 点击刷新
```

### 问题 2：样式显示异常

**可能原因**：
- CSS 变量未正确加载
- 浏览器缓存问题

**解决方案**：
```bash
# 1. 清除浏览器缓存
Ctrl + Shift + Delete

# 2. 硬刷新扩展
chrome://extensions/ → 重新加载
```

### 问题 3：JavaScript 不工作

**检查步骤**：
1. 右键点击扩展 popup
2. 选择 "检查"
3. 查看 Console 标签页
4. 查找红色错误信息

**常见错误**：
- `Uncaught ReferenceError`: 变量未定义
- `Uncaught TypeError`: 类型错误
- `Cannot read property`: 属性访问错误

### 问题 4：恢复原始版本

如果优化版本有问题，可以快速恢复：

```bash
cd d:\workspace\hr-helper
# 备份当前版本
cp popup.html popup.html.optimized
# 恢复原始版本
cp popup.html.backup popup.html
```
